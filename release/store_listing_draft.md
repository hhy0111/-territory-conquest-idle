## Full Description

Push across a dangerous frontier in short, fast roguelite runs built for portrait mobile play.

In Territory Conquest Idle, every run is a compact campaign. Capture tiles, choose your route, survive auto-battle encounters, defeat milestone bosses, and convert each loss into permanent doctrine upgrades that make the next run stronger.

Each session is designed around quick decisions and constant forward motion:

- Expand across a procedural territory board
- Choose between combat, event, utility, and boss routes
- Build stronger runs through relics and run upgrades
- Recover permanent strength through meta upgrades between runs
- Replay seeded runs with different outcomes, rewards, and risks

The current release focuses on a complete vertical slice:

- One-handed portrait UI
- Deterministic run generation
- Auto-battle combat resolution
- Run results with combat, economy, and territory summaries
- Permanent progression through essence and sigils
- Rewarded and interstitial ad support on Android

If you enjoy strategy runs, auto-battle progression, and compact roguelite sessions, Territory Conquest Idle is built to keep every run moving.
